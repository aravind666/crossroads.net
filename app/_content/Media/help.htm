Help

To save an MP3 file to your computer for later listening: If you're using a PC, right click the MP3 icon and select "Save Target As" or "Save Link As". If you're using a Mac, hold down the Control button, click the icon and select "Download Link to Disk".


If you are having problems with the quality of the sound, try downloading the message to your computer and playing it back from there. If you have questions or problems, ask for help!
